# tangible-git

**Scaffold.** Tangible Careers' git & pull-request conventions for AI agents — branch
naming, conventional commits, PR structure, a review checklist, and linking work back to
Linear.

Ships the `tangible-git` skill. The conventions are stubbed (see the skill's TODOs) and
will be filled in with the team.

## Install

Same as the other plugins in this marketplace — see the [root README](../../README.md).

- **Claude Code:** `/plugin install tangible-git@tangible`
- **Any agent:** `node plugins/tangible-linear/bin/install.mjs` links every skill in this
  repo (extend the installer to cover this plugin's skills when it graduates from scaffold).
